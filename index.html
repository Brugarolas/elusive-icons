One Font, 300 Icons

In a single collection, Elusive Icons is a pictographic language of web-related actions.

No JavaScript Required

Fewer compatibility concerns because Elusive Icons doesn't require JavaScript.

Infinite Scalability

Scalable vector graphics means every icon looks awesome at any size.

Free, as in Speech

Elusive Icons is completely free for commercial use. Check out the license.

CSS Control

Easily style icon color, size, shadow, and anything that's possible with CSS.

Perfect on Retina Displays

Elusive Icons icons are vectors, which mean they're gorgeous on high-resolution displays.

Plays Well with Others

Originally designed for Bootstrap, Elusive Icons works great with all frameworks.

Desktop Friendly

To use on the desktop or for a complete set of vectors, check out the cheatsheet.

Screen Reader Compatible

Elusive Icons won't trip up screen readers, unlike other icon fonts.
